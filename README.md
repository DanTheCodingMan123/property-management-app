# Chicken Coop

**Chicken Coop** is a property management web app where the _mother hen_ (landlord) can watch over her _eggs_ (tenants).

## What it is

Chicken Coop is a single portal for managing rentals end-to-end:

- **Landlords** manage properties, leases, rent, and maintenance.
- **Tenants** payment schedule, submit maintenance requests, and track updates.
